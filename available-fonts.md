# Available Fonts

{% tabs %}
{% tab title="Wayfinder" %}
Import with HTML / CSS:

```html
<link href="https://cdn.emmetts.dev/fonts/wayfinder/stylesheet.css" rel="stylesheet" type="text/css" />
```

Use in HTML / CSS:

```html
<style>
    .wayfinder {
        font-family: "Wayfinder CF";
        font-size: 50px;
    }
</style>
```
